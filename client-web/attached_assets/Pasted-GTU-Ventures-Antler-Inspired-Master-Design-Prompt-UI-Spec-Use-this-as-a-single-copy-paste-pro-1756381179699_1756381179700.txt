Use this as a single copy‑paste prompt for a designer, Figma plugin, or a code generator (Next.js/React + Tailwind + Framer Motion). It locks the Antler‑style aesthetic but remaps it to GTU Ventures with our brand colors, content, and IA.

0) Creative Direction (1‑liner)

Design a modern, confident, founder‑first website for GTU Ventures inspired by Antler’s aesthetic: bold typography, generous whitespace, modular card layouts, subtle depth, and refined motion. Keep it professional + warm using GTU’s purple–lavender palette.

Audience: student founders, researchers, alumni, mentors, investors, and institutional partners.

Tone: ambitious, credible, enabling. Copy is concise, benefit‑led, and action‑oriented.

1.2 Type & Scale

Font family: Display & body → Plus Jakarta Sans or Inter (fallback: system-ui, Segoe UI, Roboto, Helvetica Neue).

Headings: tight letter‑spacing for H1/H2 (‑0.015em), sentence case (avoid all‑caps except small labels).

Scale (use clamp for responsiveness):

H1: clamp(36px, 6vw, 72px) weight 800

H2: clamp(28px, 4.5vw, 48px) weight 700

H3: clamp(22px, 3vw, 32px) weight 700

Body‑L: 18px/30px; Body: 16px/28px; Caption: 14px/22px

Colors: Headings var(--ink) on light surfaces; body var(--gray-700); on dark hero use --white.

1.3 Spacing, Radius, Shadows

Spacing scale (px): 4, 8, 12, 16, 24, 32, 40, 48, 64, 80, 96.

Containers: 1280px max‑width (xl), 1440px wide hero art allowed. Side paddings: 24px mobile, 40px tablet, 64px desktop.

Radius: sm:8px, md:12px, lg:16px, xl:20px, 2xl:24px (default card), full for pills.

Shadow: shadow-sm: 0 1px 2px rgba(0,0,0,.06), shadow-md: 0 8px 24px rgba(16,12,30,.08), shadow-lg: 0 16px 48px rgba(16,12,30,.12).

Blur/Glass (for overlays & nav): background rgba(255,255,255,.7) with backdrop-blur-md.

1.4 Iconography & Illustrations

Use lucide icons. Line weight 1.5–2px. Corners slightly rounded.

Minimal geometric motifs (arrows, circles, stars) in lavender tints (10–20% opacity) as background accents.

2) Layout & Grid

12‑column grid on tablet/desktop (columns 72px, gutters 24px); 4‑column on small screens.

Generous whitespace. Prefer card‑based modules with clear vertical rhythm.

Page intro blocks use a lead paragraph and breadcrumb/eyebrow.

Section dividers: use subtle gradient strips (--grad-soft) or diagonal masks for premium feel.

3) Motion & Interaction (Framer Motion)

Easing: cubic-bezier(0.22, 1, 0.36, 1) (easeOutQuint feel).

Durations: enter 600ms; hover 150–200ms; emphasis 300–450ms.

Scroll reveal: stagger children 80ms; translate‑Y 12–16px; opacity 0 → 1.

Buttons: scale 1 → 1.03 on hover, 0.98 on press; shadow intensifies.

Cards: lift 6–10px on hover; image zoom 1.02; arrow icon slides 2–4px.

Hero BG: slow animated gradient blobs (transform: translate/scale 12–18s loop) with prefers-reduced-motion respected.

Focus: 2px outline in --gtu-lavender contrasting both light & dark.

Cards
